### Usando o CMake

Este projeto pode ser compilado usando o `CMake`, para isto, basta executá-lo dentro do diretório `compilador` dessa forma:

```bash
$ mkdir build
$ cd build
$ cmake ..
$ make
$ ./main salario2

```
